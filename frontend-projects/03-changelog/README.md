# Changelog Component

A simple and responsive **changelog component** built using **HTML** and **CSS**. This component displays a list of updates or changes made to a project, making it easy for users to stay informed about the latest improvements.

## Features

- Clean and minimal design
- Responsive layout that works on all devices
- Easy to integrate into any website
- Displays updates in a structured and readable format
