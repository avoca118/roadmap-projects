# Personal Portfolio Website - May Nyein Chan

This is a **personal portfolio website** for May Nyein Chan, a frontend developer. The website showcases projects, articles, and provides a contact form for potential collaborators or employers to reach out.

The goal of this project was to learn **CSS styling**, create **responsive layouts**, and implement **consistent typography and color schemes**.
